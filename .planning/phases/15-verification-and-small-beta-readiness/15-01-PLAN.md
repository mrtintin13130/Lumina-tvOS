# 15-01 Plan - Repeatable Build Gates

## Goal

Document and run stable local commands for the app build and test-build gates using workspace-local DerivedData.

## Commands

Generic app build:

```sh
xcodebuild -project lumina.xcodeproj -scheme lumina -destination generic/platform=tvOS -derivedDataPath .derived-data CODE_SIGNING_ALLOWED=NO -jobs 1 build
```

Generic build-for-testing:

```sh
xcodebuild -project lumina.xcodeproj -scheme lumina -destination generic/platform=tvOS -derivedDataPath .derived-data-phase15-tests CODE_SIGNING_ALLOWED=NO -jobs 1 build-for-testing
```

Optional simulator execution when CoreSimulator is healthy:

```sh
xcodebuild -project lumina.xcodeproj -scheme lumina -destination 'platform=tvOS Simulator,name=Apple TV' -derivedDataPath .derived-data-phase15-tests CODE_SIGNING_ALLOWED=NO test
```

## Acceptance

Both generic tvOS commands must complete successfully. Simulator execution is deferred when simulator services are unavailable.
