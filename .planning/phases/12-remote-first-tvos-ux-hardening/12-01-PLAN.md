# Plan

## Goal

Every primary screen and overlay has an intentional default focus target, and returning users keep focus where practical.

## Tasks

- Add stable focus identifiers to app-level tabs.
- Persist selected tab in `AppModel` so returning from overlays/playback does not disorient users.
- Add default focus to setup, server unavailable, sign-in, search, settings, detail, editorial, and loading playback.
- Keep Menu/Back behavior intact for overlays and playback.
