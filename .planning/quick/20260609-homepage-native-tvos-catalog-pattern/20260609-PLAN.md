# Quick Plan: Homepage Native tvOS Catalog Pattern

## Goal

Refine the Home page toward Apple's SwiftUI tvOS media catalog guidance, especially showing featured content above the fold while keeping shelves visible below it.

## Scope

- Keep native `TabView` app navigation.
- Adjust Home hero sizing so the first shelf peeks into the initial viewport.
- Prefer SwiftUI native button behavior with `.buttonStyle(.borderless)` and `.hoverEffect(.highlight)` where supported by the local tvOS SDK.
- Create reusable component/modifier hooks for media catalog buttons instead of repeating one-off button styling.
- Preserve real `Button` semantics for media and people cards.

- Run `git diff --check`.
