# Quick Task: Reusable tvOS Buttons

## Scope

Create reusable Lumina action button primitives so setup, search, settings, playback, and detail actions share stable tvOS sizing, icon/text layout, and focus treatment.

## Tasks

1. Add a shared button style and convenience action-row container.
2. Use single-line labels with stable min widths to reduce accidental wrapping.
3. Migrate existing non-media action buttons to the shared style.
4. Keep media cards and shelves on native `.card`/`.borderless` behavior.
5. Run a generic tvOS build if local tooling permits.
